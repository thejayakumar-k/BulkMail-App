# 📬 BulkMail App – Send Emails in Bulk with Ease

**BulkMail-App** is a lightweight Node.js-based application that allows users to send bulk emails to multiple recipients using SMTP. It's ideal for simple campaigns, internal notifications, or email functionality testing.

## 🚀 Features

- ✉️ Send bulk emails to multiple users
- 🧑‍💻 Backend-powered with Node.js and Express.js
- 📩 SMTP integration for real email delivery
- 📄 Easy-to-modify email template
- ✅ Simple UI for testing or practical use
- 💡 Ideal for learning backend email workflows

## 🔧 Technologies Used

- Node.js
- Express.js
- Nodemailer
- HTML, CSS (for UI if included)
- dotenv (for managing credentials)
